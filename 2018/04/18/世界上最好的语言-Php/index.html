Php语法

Php代码是在服务器上运行的,并转换成HTML代码发送给浏览器

php语法都包括在这个标签包裹着

echo 输出函数 相当于printf
echo ‘part 1’ . $_POST[‘name’] . ‘
‘ //点号 . 相当于将串和变量连接在一起
类似于java 里 String st = “aaa” + “bbb”;
每一句语句必须用分号 ; 结束
PHP变量名是用 $ 开头
php中 if(0 == ‘’) 结果是true
0和空串相等
empty(变量)函数 若是空则返回true
isset(变量) 函数 如果这个变量还没有设置就返回true

当然也可以将数据库一些东西写在这个共享脚步上

$_POST传送表单数据

若表单action是这个xxx.php那么在这个xxx.php文件里有个变量
$_POST,这个变量存储这个表单所有的数据,例如
有个 name="username" 的<input> 标签
那么在这个xxx.php要得到这个数据的话 $name = $_POST['username']

$_GET全局变量

GET请求中 表单数据会自动作为脚本URL的一部分发送到表单处理脚本
与$_POST类似,使用$_GET数组可以访问各种表单数据

$_GET POST区别

post请求只能从表单发出,而get请求可以打包为URL
如果页面没用用表单来启动脚本,那么他只能通过URL连接到别的
脚本,所以我们需要GET
区别
GET | 一般用于数据获取,不会有服务器有什么改变,可以直接在URL中向服务器发送数据,只适用于少量的数据
POST| 用于向服务器发送数据,从而改变服务器状态,如数据库中插入数据,只能表单

header()首部函数

这个函数可以将首部从服务器发送到服务器,不过必须要在所有的HTML代码之前调用

用header首部函数来实现用户验证

弹出的输入口令窗口可以用$_SERVER[‘PHP_AUTO_USER’] 和$_SERVER[‘PHP_AUTO_PW’]
来获取输入的密码之后再进行校正
为了增强代码复用性别的php文件可以在html开头前加上
require_once 包含这个文件这样这个文件也能用这一段代码了

也可以重定向一个页面

$_SERVER

是一个包含了诸如头信息(header)、路径(path)、以及脚本位置(script locations)等等信息的数组

  • $_SERVER[‘PHP_SELF’] 当前执行脚本的文件名

dirname(path)方法

path 参数是一个包含有指向一个文件的全路径的字符串。该函数返回去掉文件名后的目录名
dirname(“c:/test/home.php”) —-> c:/test

$_COOKIE超级全局变量用来或许一个cookie的值

setcookie(String name, Value)方法

第一个是cookie名 第二个是值作为参数传入
setcookie(‘username’, ‘value’) -> $_COOKIE(‘username’)取值
如果要删除一个cookie就得设置时间setcookie(String name, value, time)
如果要立即删除的话 time就得设置过去的时间 time()-3600 过去一小时

上传文件

表单上传后php会将文件储存在一个临时文件,如果要
改变存储位置的话 需要

move_uploaded_file(图像文件路径和文件名,目标文件位置)

  • 第一个参数一般用$_FILES[‘photo’][‘tmp_name’] 临时文件路径
  • 第二个参数自己定义

PHP中单引号和双引号的区别

    \` \\这两种转义其他都要双引号内
  • 双引号处理时会寻找变量

会话

会话开始

会话的生命调用session_start()函数告诉会话开始
调用时候这个函数首先查找是否存在一个会话ID,如果不存在就创建一个新的

$_SESSION全局变量

不同于cookie需要调用函数来赋值 可以直接 $_SESSION[‘username’] = ‘username’ 就可以
通过数组索引来使用

会话结束session_destroy()

调用session_destroy() php函数结束一个会话
如果希望用户注销之前清空会话变量,就必须手动的删除你的会话变量,
销毁一个会话的所有变量的快速方法–> $_SESSION = array();
会话其实也是用cookie来存储他的SID 如果要结束会话也得把这个cookie清除
用isset($_COOKIE[session_name()]) 来判断是否有会话的cookie
如果有就用setcookie清除掉
如果禁用cookie的话可以通过URL来传入 可以用SID全局变量
