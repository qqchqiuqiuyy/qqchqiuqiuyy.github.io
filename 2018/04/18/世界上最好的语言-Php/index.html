Posted on 2018-04-18

Php语法

Php代码是在服务器上运行的,并转换成HTML代码发送给浏览器

php语法都包括在这个标签包裹着

echo 输出函数 相当于printf
echo ‘part 1’ . $_POST[‘name’] . ‘
‘ //点号 . 相当于将串和变量连接在一起
类似于java 里 String st = “aaa” + “bbb”;
每一句语句必须用分号 ; 结束
PHP变量名是用 $ 开头
php中 if(0 == ‘’) 结果是true
0和空串相等
empty(变量)函数 若是空则返回true
isset(变量) 函数 如果这个变量还没有设置就返回true

foreach函数

将一些复选框的值存入一个数组中

$_POST传送表单数据

若表单action是这个xxx.php那么在这个xxx.php文件里有个变量
$_POST,这个变量存储这个表单所有的数据,例如
有个 name="username" 的<input> 标签
那么在这个xxx.php要得到这个数据的话 $name = $_POST['username']

$_SERVER

是一个包含了诸如头信息(header)、路径(path)、以及脚本位置(script locations)等等信息的数组

  • $_SERVER[‘PHP_SELF’] 当前执行脚本的文件名

PHP中单引号和双引号的区别

  • 单引号被认为是原始文本 \n等转义字符不可用 只支持
    \` \\这两种转义其他都要双引号内
  • 双引号处理时会寻找变量

    在 Apache安装路径\htdocs下 创建php文件

  1. $name = $_POST[‘name’] ; $pwd = $_POST[‘pwd’]
