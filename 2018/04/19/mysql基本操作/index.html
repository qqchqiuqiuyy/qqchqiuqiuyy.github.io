mysql基本操作

Posted on 2018-04-19

mysql一些语法

在cmd之中的操作

登陆mysql

mysql -uroot -p

显示所有的数据库

show databases;

使用某个数据库

use xxx;

新建数据库

create databases xxx;

显示这个数据库的所有表

show tables;

删除这个数据库

drop database xxx;

使用这个表

use xxx;

不修改表的数据情况下给表添加一个新列

ALTER TABLE table_name ADD column_name column_type

不修改表的数据情况下给表删除一个新列

ALTER TABLE table_name DROP column 列名

修改一列的列名和数据类型

ALTER TABLE table_name CHANGE COLUMN column 列名 类型

NOW() 函数 可以得到当前日期/时间

显示表中所有数据SELECT * FROM 表名
